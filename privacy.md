# Privacy Policy for HN Reader

**Last updated: June 8, 2026**

HN Reader is a client for reading Hacker News content. This app:

- **Does not collect** any personal information, accounts, or analytics
- **Does not track** your reading behavior or browsing history
- **Stores preferences locally** (theme, font size, visited articles) on your device only — this data never leaves your device
- **Makes network requests** to Hacker News (news.ycombinator.com), the HN Firebase API (hacker-news.firebaseio.com), and Algolia (hn.algolia.com) to fetch story and comment content. These are necessary for the app to function. These services may log standard HTTP request metadata (IP address, user agent) per their own privacy policies.

## Third-party services

The app communicates with the following services to display content:

- **news.ycombinator.com** — Hacker News website, used for comment ordering
- **hacker-news.firebaseio.com** — Official HN API for story data
- **hn.algolia.com** — HN search API for comment data

None of these services receive any personal information from this app beyond what a normal web browser would send.
